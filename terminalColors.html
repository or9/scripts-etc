Setting terminal colors on Mac OS X

cd ~
vim .bash_profile
# If file existed already, add to bottom; otherwise file will be created from previous command.
# Place following commands at end
export CLICOLOR=1
export LSCOLORS=ExFxBxDxCxegedabagacad
CLICOLOR
[1|0]; Boolean indicating whether to use command line colors.
LSCOLORS
The value of this variable describes what color to use for which attribute when colors are enabled with CLICOLOR.
This string is a concatenation of pairs of the format fb, where f is the foreground color and b is the background color.
